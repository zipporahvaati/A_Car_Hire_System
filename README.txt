# Car Hiring System

A **full-stack car hiring web application** built with **Flask** and **PostgreSQL**.  
This system allows admins to manage cars and customers to rent and return cars in real-time.  

---

## Features

- **Admin Dashboard**  
  - Add, view, and manage cars  
  - Track car availability  

- **Customer Dashboard**  
  - Browse available cars  
  - Rent and return cars  
  - View currently rented cars  

- **Backend**: Flask & SQLAlchemy  
- **Database**: PostgreSQL  
- **Frontend**: HTML & CSS templates
